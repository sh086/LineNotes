      # Jenkins

      ​ 　Jenkins 是一个开源软件项目，是基于 Java 开发的一种持续集成工具，用于监控持续重复的工作，旨在提供一个开放易用的软件平台，使软件的持续集成变成可能。

      # Jenkins Dokcer

      ​ 　首先，编写docker-compose.yml 配置文件。

      version: '3.1'
services:
  jenkins:
    restart: always
    image: jenkinsci/jenkins
    container_name: jenkins
    ports:
      # 发布端口
      - 8080:8080
      # 基于 JNLP 的 Jenkins 代理通过 TCP 端口 50000 与 Jenkins master 进行通信
      - 50000:50000
    environment:
      TZ: Asia/Shanghai
    volumes:
      - ./data:/var/jenkins_home

      # Jenkins配置

      （1）配置 JDK & Maven

      ​ 　首先，将上传 JDK 和 Maven 的 tar 包到服务器（容器数据卷目录），并解压到Jenkins数据卷目录下。

      ./data/jdk1.8.0_152
./data/apache-maven-3.5.3
      1
      2

      ​ 　然后，进入Manage Jenkins -> Global Tool Configuration，配置JAVA_HOM路径，然后，点击保存。

      ​ 　接着，配置 MAVEN_HOME路径，然后，点击保存。

      （2）配置本地化（显示中文）

      ​ 　首先，安装 Locale 插件，安装成功后，在Manage Jenkins -> Configure System -> Locale选择中文即可。
